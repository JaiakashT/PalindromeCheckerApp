# 🔁 Palindrome Checker App

## 📌 Overview
The **Palindrome Checker App** is a Java-based console application that validates whether a given string is a palindrome using multiple approaches.

This project is designed to demonstrate how **Core Java concepts and Data Structures** can be applied to solve a simple problem in different optimized ways.

Each use case introduces a new technique, evolving from basic string checks to advanced design patterns and performance analysis.

---

## 🎯 Objective
The objective of this project is to:

- Understand different ways to solve a palindrome problem
- Learn how data structures impact performance
- Apply object-oriented design principles
- Compare algorithm efficiency
- Build clean and modular Java programs

---

## 🧠 Concepts Covered

### 🔹 Core Java
- Classes & Objects
- Methods & Control Flow
- String Handling
- Recursion
- Exception-free logic design

### 🔹 Data Structures
- String & Character Array
- Stack (LIFO)
- Deque (Double-ended queue)
- Linked List

### 🔹 Advanced Concepts
- Two-pointer technique
- Recursion & Call Stack
- Strategy Design Pattern
- Performance Measurement using `System.nanoTime()`
- Input preprocessing using Regex

---
